# gym-website

중고 헬스용품 거래 사이트 + 헬스장 찾는 웹사이트

## 🖥️프로젝트 소개
중고 헬스용품 거래 사이트 + 헬스장 찾는 웹사이트 (번개장터+다짐)

### ⚙️개발환경
java 8
Intellij
Springboot(2.7)
Mysql(5.x)

### 🔧주요기능

#### 로그인
- ID and PW로 검색 -> 있다면 로그인 성공
- 로그인시 세션에 userID 생성

#### 회원가입
- 아이디 중복 체크 -> 아이디 검색해서 있다면 X
- 비밀번호 재확인

#### 상품등록
- 상품 올리기
- 부위, 브랜드, 가격 별로 검색 가능

#### 장바구니
- 장바구니 개수 확인 가능
- 이미 장바구니에 있다면 버튼 색 다르게 -> 이미 장바구니에 있는 상태에서 한번 더 누르면 장바구니에서 삭제
